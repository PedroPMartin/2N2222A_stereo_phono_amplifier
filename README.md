2N2222A Stereo Phono Amplifier
=======
Based on [Hans Polak and Ivey's](https://www.diyaudio.com/community/threads/2n2222a-phono-preamp.158918/post-6392737) design. <br>
Thanks for Hans Polak and Ivey.
